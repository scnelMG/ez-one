# Chrome Extension

Chrome Extension for job posting extraction, preview, and save.

## Responsibilities

- Check service login/session state
- Extract job posting data from supported job sites
- Preview extracted data before saving
- Save job postings to basket through backend API
- Report extension events for debugging and quality tracking

## P2 / Reserved Responsibilities

- Fetch document input information for input assistance
- Automatically fill external application forms
- Handle failed input fields with copy/download fallback
